## 📁 폴더 구조 
```javascript
📦src
 ┣ 📂assets
 ┣ 📂components // 여러 페이지에서 사용하는 컴포넌트
 ┣ 📂css // 스타일링
 ┣ 📂data // 구 좌표, 사고 유형 등 데이터
 ┣ 📂pages // 페이지
 ┣ 📂redux // redux reducer 정의
 ┣ 📂utils // 유틸리티 함수
 ┣ 📜App.jsx
 ┣ 📜index.js
 ┣ 📜reportWebVitals.js
 ┣ 📜Router.js // Routing
 ┗ 📜setupTests.js
```

$ cd bbiyong-front
```

### Frontend
```
$ npm install
$ npm start
